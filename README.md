# CGPA Calculator (IPU Pattern)

A simple and accurate web-based CGPA calculator for **Students of Guru Gobind Singh Indraprastha University (GGSIPU)**.

This calculator follows the **official IPU grading system (Ordinance 11)** and helps students calculate their CGPA using:
- Internal marks
- End-semester marks
- Official subject credits
